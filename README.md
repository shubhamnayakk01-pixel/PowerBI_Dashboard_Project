## 📊 Lokmanya Co-operative Society Dashboard

## Overview
An interactive Business Intelligence dashboard built to analyze and forecast financial data for a co-operative society bank.  
The dashboard provides insights into loans, investments insurance, and customer distribution, along with a 30-day loan forecasting model.

---

## Key Features
- KPI tracking (Loan, Investment, Insurance, Mutual Funds, Locker)
- Loan, Insurance and Investment distribution analysis
- Month-on-Month trend analysis
- 30-day Loan Forecasting
- State-wise and Branch-wise customer insights
- Interactive filters for dynamic analysis

---

## Tech Stack
- Power BI
- DAX
- Excel

---

## Key Insights
- Maharashtra has the highest customer base
- Home and Education loans dominate the portfolio
- Equity and Index funds lead investments
- Loan activity peaks mid-year
- Forecast shows moderate growth trend

---

## How to Use
1. Download the '.pbix' file  
2. Open in Power BI  
3. Use filters to explore insights
